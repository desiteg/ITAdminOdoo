Punto de venta adaptado para facturar electrónica en México (CFDI 2017)

Adapta el terminal punto de venta para realizar facturas electrónicas en México (CFDI 2017)

Se pueden emitir facturas:

  • Por venta
  • De ventas de un periodo (día, semanas, mes)
  • 1) Con un solo concepto
  • 2) De todas los productos


  • See Pack : we can see pack here.
  • 1) from sale->product pack


  • Sale order : user can add whole pack in sale order once it is confirm sale order you can see delivery option


  • Delivery screen: You just have to select one pack for sale order and related products for this pack will be automatically added in delivery screen.


Additional Features

  • Add Bundle in sale order using Button.


  • Product Bundle Filter in product kanban view.
